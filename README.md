<h1>Taskify</h1>

A task management dashboard built with React.js and PocketBase.
🚀 Tech Stack

React.js
React Router DOM
Context API
React Hook Form
PocketBase
Recharts
date-fns
CSS Modules

<h3>✨ Features</h3>

Create, edit, and delete tasks
Filter tasks by status, priority, and search
Completion rate chart
Tasks overview chart
Priority distribution chart
Context API for state management

<h3>🛠️ Getting Started</h3>
Prerequisites

Node.js
PocketBase v0.29.0

<h4>Installation</h4>

Clone the repo

bashgit clone https://github.com/KamyabT/taskify

Install dependencies

bashnpm install

Download PocketBase v0.29.0 from here
Start PocketBase

bash./pocketbase serve

Start the app

bashnpm start
