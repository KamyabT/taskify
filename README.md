# Taskify

A task management app built with React and PocketBase.

## Live Demo
Coming soon

## Features
- Create, edit, and delete tasks
- Filter tasks by status, priority, and search
- Completion rate chart
- Tasks overview chart
- Priority distribution chart
- Context API for state management

## Tech Stack
- React.js
- PocketBase
- Context API
- React Hook Form
- Recharts
- date-fns
- CSS Modules

## Getting Started

### Prerequisites
- Node.js
- PocketBase v0.29.0

### Installation
1. Clone the repo
   git clone https://github.com/KamyabT/taskify

2. Install dependencies
   npm install

3. Download PocketBase v0.29.0
   https://github.com/pocketbase/pocketbase/releases/tag/v0.29.0

4. Start PocketBase
   ./pocketbase serve

5. Start the app
   npm start
